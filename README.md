# Byod Bot

## Current supported reverse proxies:
- [x] Caddy
- [ ] Nginx
- [ ] Apache

## Description
A discord bot allowing anyone to implement BYOD (bring your own domain) into their service.

## Setup

### Requirements
- Ruby 3.0.0 or higher
- Bundler
- PostgreSQL server
- Discord bot token
Or use the Docker image to skip:
- Ruby
- Bundler
- PostgreSQL server
